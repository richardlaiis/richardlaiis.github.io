總共 25+1 學分 (1 是服學)，我已經死掉了。有些必修的評分方式或是教學範圍我就不提了，因為網路上應該已經有很多資訊。

數值方法

範圍

請直接參見課程網頁： https://www.csie.ntu.edu.tw/~cjlin/courses/nm2026/

心得

林智仁教授的上課方式是上課放已經錄好的影片，偶爾暫停下來講幾句，老實說我非常不喜歡這種模式，因為我覺得很容易睡著 XD。這堂課的內容大概就是講電腦如何更有效率的處理線性系統、浮點數系統，個人認為最困難的 part 大概就是 Conjugate Gradient method 跟 Fast fourier transform，但老實說沒有完全懂還是能考試，這就要講到這堂課的考試方式了。

考試可以 Open Book，基本上什麼都可以帶，所以我基本上把課程的所有簡報和五年的考古題都印下來，個人認為非常好用w，需要額外具備的知識大概就是線性代數與微積分。作業部份會需要寫一點 C/Matlab 的程式，也蠻簡單的。

整體來說我覺得這堂課是門好課，可以學到非常多東西，簡報編排的相當完整，甚至不上課也能看懂上課在幹嘛。

智慧型汽車導論

範圍

Timing analysis, system design, intersection management, verification 還有很多 high-level 的東西。

心得

有兩份作業，一個期中，一份 Final project。

上完這門課我最大的感想就是 autonomous driving 真是太難實現了，需要顧慮的事情超爆多。

這門課的含括了許多 high level 的觀念，雖然只是很粗淺的講過，但也有一些很實用的觀念 可以應用在自駕車以外的領域，例如 v-model, MILP, verification。當然也有一些比較需要計算的像是 timing analysis, graph based-model intersection management。總之是門好課可以修！

數位系統與實驗

兩份各佔 5 % 的作業，剩下全看考試。

範圍

Combinational logic, sequential logic, state machine design 等等。

心得

也是被擠壓到時間的一科，期考自己沒有細心吃了很多虧，只能說希望自己最後有 A。有人說這門課是計概 2.0，但我覺得可以吃老本的頂多到 Karnaugh Map 那個章節，後面 (state machine, hazards…etc) 基本上都是新學習的知識。

作業系統

沒什麼心得，這學期修的學分太多了，導致這一科的時間被擠壓到蠻多，我完成 machine problem 都有用 AI 輔助 QwQ 兩次期考的成績也慘不忍賭。

老師教的很好，全是我的問題。😭

Web APP 程式設計與應用

範圍

Django, AI 工具的使用

心得

基本上就是 LLM 課，我相信每個修課的人都使用 copilot 或 claude code 的來完成專案 (問過蠻多組ㄉ)

我們組做了兩個專案，一個是共享記帳系統，一個是仿 Discord 的社群軟體: Discoord。其實我覺得現在的數位工具市場其實有些飽和了，就是你大多數做的東西不會比熱門的商用/開源軟體好用，只是在重複造輪子，我們這組就是一個很好的例子。不過很高興這學期分到的組員每個都不擺爛，不像上學期影像處理的組員 :(

代數導論二

範圍

Field theory, Galois theory, Function field, solvability, basic of kummer theory 等等。

心得

其實我覺得自己並沒有像那些數學系的人一樣，對數學充滿熱情。我大多時候就是看看課本、講義，開始寫作業，發現自己都不會，最後跟 Gemini 開始激烈討論 ;-;

我感覺自己只是一個理解者，稍微碰到一絲抽象的概念就再也沒有靈感寫出題目ㄌ。之後我應該就不會再修數學系的課了，專心在資工的本業比較划算 QQ

那這堂課怎麼樣呢？我覺得佐籐信夫蠻會上課的，舉了非常多實例，讓本來很抽象的東西變得比較容易理解。不過這學期的作業題目變好少，內容也跟上課重疊不少 QQ 但有提供 Lecture notes 超讚。整體而言是門好課，對於想入門抽象代數與迦羅瓦理論的人非常適合，雖然是英文授課但老師口音很標準，唯一要小心的是助教課飆車飆很快，內容在期考也有機會考出來。

重量訓練

梁澤敬老師是臺灣前 100 公尺紀錄保持人，超猛。

這堂課沒什麼好說的，推爆。教練會從零開始教重量訓練的各種動作，不管核心、上肢、下肢都可以練到，而且也可以學到如何安排自己的訓練菜單 (肥大、爆發力之類的)。此外，作業跟考試也很水，大部分的分數都可以靠點名賺到，是一門學得到東西又很甜的課，超讚～

這們課也讓我開始正視自己糟糕的身材…

機率

洪一平的授課方式不知道為什麼總是有些玄學的氣質 XD

老師上課很慢，簡報很醜，所以我很後悔自己沒有全程閱讀 textbook 來跟上進度，但兩次期考對我來說有點太難了 QQQQQ，希望我能夠通過 NFT 加分作業彌補自己的愚蠢 ><

課程以外我學到…

  1. 世界上沒有這麼多觀眾 (但有很多檢核你能力的裁定者)
  2. 修太多學分沒有任何意義，學習的 quality 比較重要 (希望自己以後不會再面臨到一個禮拜 7 個考試、2 個報告的窘境)
  3. 休息時好好休息，學習時絕對要保持專注
